📥 INSTALLATION & SETUP

Step 1: Boot the System

1. Open (BOOT THIS FIRST) BIOS.html in your browser 2. Wait for auto-boot or press any key 3. (Optional) Press F+1 to view BIOS settings

Step 2: Access Desktop

System automatically boots to Desktop.html

Step 3: Enable Performance Features

1. Click "Performance" application 2. Click "ENABLE GAME BOOST" button 3. Performance optimizations are now active (+50%)

Step 4: Monitor System

Click "Task Manager" to monitor processes and system health

✅ Tips: For best performance, close unnecessary applications before playing games. Use Task Manager to monitor system resources.
